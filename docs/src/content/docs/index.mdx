# NZOTH

 **N**est + **Z**od + **O**penAPI + **T**yped + **H**elpers (pronounced n'zoth)

Build fully typed REST APIs with NestJS, Zod, and OpenAPI.

- Type-safe validation (requests, bodies, responses)
- Helpful decorators (`TypedRoute`, `TypedBody`, `TypedParam`, `TypedQuery`)
- Pagination, sorting, and filtering ready to use
- OpenAPI schema generation from Zod

Head to "Getting Started" to jump in quickly.

Built on top of:

- [NestJS](https://github.com/nestjs/nest)
- [Zod](https://github.com/colinhacks/zod)
- [zod-openapi](https://github.com/samchungy/zod-openapi)
